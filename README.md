### 🛠️ Featured project

**[WinASys — Windows Forensics Automation Tool](https://github.com/makhentosch/WinASys-WindowsForensicsAutomationTool)**
A Python CLI that automates collection and parsing of key Windows forensic artifacts — ShellBags, ShimCache, AmCache, Prefetch, Jump Lists, LNK, UserAssist, USB history — cutting manual triage from hours to minutes. Wraps the Eric Zimmerman toolset, registry analysis, and SQLite parsing behind a single menu-driven interface.
